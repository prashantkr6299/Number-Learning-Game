# Number-Learning-Game
An interactive and engaging web-based learning game designed to help children practice number recognition, counting, reverse counting, and listening skills through voice-based quizzes.  This project is built using HTML, CSS, and JavaScript and works directly in the browser without any external libraries.

🚀 Features
👤 Individual Mode

Enter player name

Set maximum number range

Forward counting practice

Reverse counting practice

Voice-based quiz

Instant score tracking

Motivational appreciation messages

Celebration animations 🎈⭐

🏫 Classroom Mode

Add multiple students

Set number of quiz questions per student

Individual quiz for each student

Automatic score calculation

Final results table

Top performer detection 🏆

🔊 Voice Support

Uses Web Speech API (SpeechSynthesis)

Speaks numbers aloud

"Listen Again" feature in quiz mode

🎨 Interactive UI

Color-changing numbers

Smooth animations

Floating stars and balloons

Responsive design (Mobile + Desktop)

🛠️ Technologies Used

HTML5

CSS3 (Flexbox, Animations, Responsive Design)

Vanilla JavaScript

Web Speech API

No external frameworks or libraries used.
